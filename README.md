# VIT FFCS Planner

A fast, no-login **VIT FFCS Planner** to build your timetable instantly.

👉 Live tool: https://vit-ffcs-planner.pages.dev/

## What this does

* Build VIT timetables instantly
* Test FFCS slot combinations
* Avoid clashes in real time
* No login or signup required

## Why this exists

Most existing VIT FFCS planners are slow or cluttered.
This tool is designed as a minimal, fast scratchpad for slot selection.

## Keywords

VIT FFCS planner, VIT timetable planner, FFCS slot combinations, VIT slot planner

## How to use

1. Click on a cell
2. Select a slot combination
3. Add subject names (optional)
4. Build your timetable quickly

## Tech

* Vanilla JS
* No frameworks
* Static deployment
